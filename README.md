# Simple Shopify Theme

Ця тема показує процес створення теми для платформи Shopify покроково. З самого мінімому, до повноцінної системи.

Весь опис, назви комітів та коментарі в коді написані українською, з метою полегшення вивчення  носіям української.

## Дизайн

В роботі на цим проєктом я вирішив використати [дизайн](https://www.figma.com/community/file/999496395784541518/e-commerce-ui-kit) з ком'юніті Figma з відкритою ліцензією.

Дякую автору цього дизайну за можливість попрактикуватись.

## Установка та перевіка теми

Для роботи із темою нам потрібно встановити інструменти [Shopify CLI](https://shopify.dev/docs/api/shopify-cli), як це зробити доступно розказано в [певному розділі документації](https://shopify.dev/docs/api/shopify-cli#installation)

Потім зклонував репозиторій, заходимо в папку з темою, та виконуємо:

```sh
shopify theme dev
```

Відкриваємо сторінку посилання, і дивимось на нашу тему.
